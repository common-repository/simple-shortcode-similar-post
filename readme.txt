== Description ==

This plugin is a better way to set a similar post in article.
All you have to do is setting a shortcode.This plugin works setting similar post.

== Installation ==

1. Upload the `simple-shortcode-similar-post` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Embed `[sssp post_num=1]` on your post.
